#Pet adoption website
A full‑stack web application for managing and promoting pet adoption. Users can browse pets, and authenticated users can create, edit, and manage animal listings.

##Features

- Browse available pets for adoption
- User authentication (signup / login)
- Create, edit, and delete pets (only by the creator)
- Detailed pet profiles (age, gender, weight, type, etc.)
- Status management (available / adopted / reserved)
- Responsive UI
